ปัญหาเดิมของ VR Treadmill

ลู่วิ่ง VR แบบเดิมมักต้องใช้กรงใหญ่ ราคาแพง ติดตั้งยุ่งยาก หรือใช้ชุดสวมพิเศษ ทำให้ผู้เล่นถูกจำกัดการเคลื่อนไหว

Free Step แก้ด้วย

  • ระบบเซนเซอร์หลายจุด (distance sensors, IMU, encoder) ตรวจจับตำแหน่งผู้เล่นแบบเรียลไทม์
  • ควบคุมความเร็วสายพานด้วย PID ลดการสั่นและกระตุก
  • ดีไซน์กะทัดรัด ลดความจำเป็นต้องมีกรงล้อม
เซนเซอร์

VL53 / VL53L1X ตรวจขอบ, MPU6050/MPU9250 ตรวจองศา, Optical encoder สำหรับตำแหน่ง

การควบคุม

PID speed & position control, ESP32 / STM for MCU, MQTT / Node-RED สำหรับ dashboard

รองรับ VR

เชื่อมต่อกับ SteamVR หรือ Pico (ผ่าน PC) — ปรับ mapping เป็น keyboard/mouse input หรือ vJoy

Demo & Timeline

ตัวอย่างการทำงานเบื้องต้น และแผนพัฒนาเป็นขั้นตอน

  1. Phase 1 — PID speed control (เสร็จแล้ว)
  2. Phase 2 — Position control ด้วย VL53 array + MPU (กำลังพัฒนา)
  3. Phase 3 — Integration กับ SteamVR + UX polishing
